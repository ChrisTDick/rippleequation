---------------------------

This folder contains three subfolders. Each subfolder contains .py file. If run, this .py file will execute and evaluate one version of the model implemented for this project. All necessary data is included in the folder in pickle, json or csv format. This project was made in Python 3.7. Modules required to run the .py files are:

NetworkX 2.2
TextBlob 0.15.3
SpaCy 2.1.3
Pandas 0.23.4
Numpy 1.15.2
